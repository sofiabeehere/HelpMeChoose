PROJECT DESCRIPTION

This interactive prototype takes three inputs: temperature, force, and touch, to drive a motor and turn LEDs on and off. The final implementation is meant to be enclosed in a coffee mug, to automatically stir beverages at the touch of a button.

PROJECT INFORMATION

School: Simon Fraser University Type of Project: Group/Team
Faculty: Communication, Art & Technology Grade Received: A
Course: IAT 267 Tools Used Arduino, Processing
Completion Date: Spring 2017

PROJECT REQUIREMENTS

The project idea should address a subject related to the use of sensors and Arduino and Processing in an application. The project work includes idea development as well as practical implementation.

Projects must include the following:

  • Arduino
  • Communication with computer: Processing application running on the computer is required
  • At least three sensors of a different kind
  • Motion or sound
  • A graphical interface for user interaction or on-screen visualization

PROJECT PROCESS, GOAL, AND RESULT

The goal of the final project was to develop a physically interactive system that takes an input of your choice (or multiple inputs) and translates this input via a computer system (microcontroller / laptop / desktop computer) into a meaningful/surprising/critical/artistically challenging output.

RECOMMENDATIONS & SUGGESTIONS

Start planning early! Clearly identify what components you'll need to complete your project in advance and purchase multiples in case if anything breaks.
